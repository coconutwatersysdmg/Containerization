# 机械臂吸盘角点和方向字段说明

本文档说明装箱结果文件中与机械臂吸附姿态相关的字段。

## 坐标约定

每个箱子的 `position` 表示箱体在托盘坐标系中的最小角点：

- `position.x`: 箱体 x 方向最小坐标
- `position.y`: 箱体 y 方向最小坐标
- `position.z`: 箱体 z 方向最小坐标，也就是箱体底面高度

箱体尺寸字段含义：

- `length`: 箱体沿 x 方向的尺寸
- `width`: 箱体沿 y 方向的尺寸
- `height`: 箱体沿 z 方向的尺寸

箱体顶面高度为：

```text
top_z = position.z + height
```

## 新增字段

### `suction_box_corner`

表示吸盘角点对准的箱子顶面角点。

可选值：

| 值 | 箱子角点坐标 |
| --- | --- |
| `x_min_y_min` | `(position.x, position.y, top_z)` |
| `x_max_y_min` | `(position.x + length, position.y, top_z)` |
| `x_min_y_max` | `(position.x, position.y + width, top_z)` |
| `x_max_y_max` | `(position.x + length, position.y + width, top_z)` |

### `suction_cup_corner`

表示与箱子角点对齐的吸盘角点。

当前算法中该值与 `suction_box_corner` 一致。控制端可按同名角点理解：

- `x_min_y_min`: 吸盘的 x 最小、y 最小角点
- `x_max_y_min`: 吸盘的 x 最大、y 最小角点
- `x_min_y_max`: 吸盘的 x 最小、y 最大角点
- `x_max_y_max`: 吸盘的 x 最大、y 最大角点

### `suction_orientation`

表示吸盘在托盘 x/y 坐标系中的方向。

可选值：

| 值 | 含义 |
| --- | --- |
| `cup_400x_600y` | 吸盘 400mm 边沿 x 方向，600mm 边沿 y 方向 |
| `cup_600x_400y` | 吸盘 600mm 边沿 x 方向，400mm 边沿 y 方向 |

如果机械臂硬件不允许吸盘绕 z 轴旋转 90 度，应在算法参数中关闭 `allow_suction_rotation_90`。

### `suction_cup_x_size` / `suction_cup_y_size`

表示当前姿态下吸盘沿托盘 x/y 方向的尺寸。

控制端应优先使用这两个字段，而不是从 `suction_orientation` 字符串中解析尺寸。

### `suction_rect_x_min` / `suction_rect_x_max` / `suction_rect_y_min` / `suction_rect_y_max`

表示当前吸盘姿态在托盘 XY 平面上的投影范围。

这些字段主要用于：

- 控制端调试
- 离线复核机械臂可达性
- 可视化吸盘外伸区域

## 控制端使用步骤

1. 读取箱子的 `position`、`length`、`width`、`height`。
2. 根据 `suction_box_corner` 计算箱子顶面吸附角点坐标。
3. 根据 `suction_cup_corner` 将吸盘对应角点对准该箱子角点。
4. 根据 `suction_cup_x_size` 和 `suction_cup_y_size` 设置吸盘在 x/y 方向的长宽。
5. 从高处沿 z 方向竖直下降，将箱子放置到 `position` 对应坐标。

## 示例

若结果文件中某个箱子为：

```text
position.x = 100
position.y = 200
position.z = 300
length = 500
width = 350
height = 250
suction_box_corner = x_max_y_min
suction_cup_corner = x_max_y_min
suction_cup_x_size = 400
suction_cup_y_size = 600
```

则箱子顶面吸附角点为：

```text
(position.x + length, position.y, position.z + height)
= (600, 200, 550)
```

控制端应让吸盘的 `x_max_y_min` 角点对准 `(600, 200, 550)`，吸盘 400mm 边沿 x 方向，600mm 边沿 y 方向。
